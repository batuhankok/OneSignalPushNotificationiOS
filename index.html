Hey!

I know, setting up the Push Notification can be so annoying problem for iOS developers. However, OneSignal makes your life easy for sending notifications by using web interface of OneSignal. The installing process is a little bit hard, so, I have written here to make your life easier!

  1. Go to onesignal.com
  2. Login and click the "Create a new app"
  3. Select "Apple iOS (APNS)"
  4. Open new tab, go to onesignal.com/provisionator
  5. Click Get Started
  6. Login with your Apple Developer Account
  7. Select your app that you want to use push notification, click Generate
  8. Click Download .p12 File and copy the password
  9. Turn back to the onesignal.com's tab
  10. Select the file that you have just downloaded
  11. Paste the password to the related input, click Save
  12. Select Native iOS, then copy Your App ID (like: 5eb5a37e-b458-11e3-ac11-000c2940e62c)
  13. You can close the web browser
  14. Open the terminal (hint: you can push CMD + Space and write terminal...)
  15. Write, cd and drag drop the your app folder to the terminal, press enter
  16. It has to seem like this: cd /Users/***/Desktop/XcodeApps/***\ ***
  17. Write, pod init, then press enter
  18. Write, open Podfile, then press enter
  19. Write between do - and, pod 'OneSignal' to the file that did opened and save it
  20. Write, pod install, then press enter
  21. Then open your project with clicking example.xcworkspace
  22. Open Capabilities tab, Push capability: switch on
  23. Background Capability: switch on
  24. In the Background Capability, remote push notification: switch on
  25. Then, edit your AppDelegate.swift like above:
    26. ```swift import OneSignal func application(application: UIApplication, didFinishLaunchingWithOptions launchOptions: [NSObject: AnyObject]?) -> Bool { //Add this line. Replace '5eb5a37e-b458-11e3-ac11-000c2940e62c' with your OneSignal App ID (you have just copied.) OneSignal.initWithLaunchOptions(launchOptions, appId: "5eb5a37e-b458-11e3-ac11-000c2940e62c") return true } ```
  26. Then, save your Xcode project.
